----------------------
Compile Project code
----------------------

Example: C:\User\rithv\workspace1\MSD\src>make

C:\User\rithv\workspace1\MSD\src-It is the path to the directory 
which contains the .cpp file

make- it is the command used to compile the .cpp file

There is also a command "make clean" which is used to remove the 
executable file.


----------------------
Run Project code
----------------------

Example: C:\User\rithv\workspace1\MSD\src>MSD.exe 0 G:\traceFile.txt

MSD.exe- It is created after we compile the corresponding .cpp file

0- It is a command line argument which is used to assign the 
mode variable 

G:\traceFile.txt- It is the path of the trace file
